Welcome to mtcars Analysis shinyapp.


This web application is designed to show some basic data analysis over the R dataset, mtcars.  The application contains several panel tabs above for navigating between the analysis pages.  The four panels are Info (this page), Dataset which explores the detailed dataset, Correlation which shows a correlation chart for the variables you choice and a Regression panel used to calculate the R-Squared for a formula you specify.

The mtcars dataset contains data from 1974 Motor Trend US magazine.  It comprises of data from 32 automobiles from 1973–74 models.  The dataset contains the following columns.

mpgMiles/(US) gallon
cylNumber of cylinders
dispDisplacement (cu.in.)
hpGross horsepower
dratRear axle ratio
wtWeight (1000 lbs)
qsec1/4 mile time
vsV/S
amTransmission (0 = automatic, 1 = manual)
gearNumber of forward gears
carbNumber of carburetors
